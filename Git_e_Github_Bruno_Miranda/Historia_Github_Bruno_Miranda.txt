a.	Quem inventou o sistema de controle de versão Git e por quê?
R: Linus Torvalds, pq ele estava muito pistola Quando o Bitkeeper desligou seu serviço gratuito, Linus Torvalds criou o Git por conta própria porque nenhum dos outros sistemas de controle de versão atendia às suas necessidades.

b.	Quem atualmente é o dono do Github e por quê?
R: Microsoft, já que ela é companhia privada que mais colabora com o GitHub a compra foi essencial para concentrar seus esforços nas quatro principais frentes que Nadella considera cruciais: Nuvem, IA, Serviços (incluso Windows, Office e Xbox) e mercado corporativo.

c.	Existe outra maneira, além do terminal, de trabalhar com o Github?
R: Sim, pelo VSCODE e pelo Github de Desktop.
